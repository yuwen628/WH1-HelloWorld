# Hollow World 🌌

A premium, aesthetically pleasing, and highly interactive frontend showcase demonstrating hollow typography and glassmorphism design.

## ✨ Features

- **Hollow stroke typography**: The main heading utilizes native CSS `-webkit-text-stroke` to create a beautiful, transparent title with a glowing pulse animation.
- **Interactive 3D Tilt**: The central container responds to mouse movements, tilting dynamically to follow the cursor.
- **Ambient Particle Engine**: Lightweight, vanilla JavaScript-powered floating particles drift in the background with customizable colors and burst effects.
- **Glassmorphic UI**: High-fidelity blurred backdrop cards utilizing HSL-based color tokens, dark mode gradients, and soft border highlights.

## 🚀 How to Run

No build step or installations required!

1. Clone this repository:
   ```bash
   git clone https://github.com/tcnr-12/WH1-HelloWorld.git
   ```
2. Navigate into the project folder:
   ```bash
   cd WH1-HelloWorld
   ```
3. Open `index.html` directly in your favorite web browser, or launch it with a local server:
   - For VS Code users: Use the **Live Server** extension.
   - For python users: Run `python -m http.server 8000` and visit `http://localhost:8000`.

## 🛠️ Built With

- **HTML5** & **CSS3** (Vanilla Variables & Keyframe Animations)
- **JavaScript ES6** (Dynamic DOM manipulation & event handlers)
